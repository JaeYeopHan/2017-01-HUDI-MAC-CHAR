# 개발환경

* Server Function
    * BuildTool: Gradle

    * Database: MySQL

    * Spring Websocket
* Front-End Function
    * HTML CSS jQuery (ES6, WebPack 모듈화)
    * BootStrap
    * Template Engine: Handlebars
* IOS
    * Xcode
* Android
    * Android Studio
